Research

Trojan Attacks on Machine Learning Models: May 2022 - December 2023

Summary: The viability of Trojan attacks on Nuclear based neural networks was determined. This project required creating a neural network for transient classification and then implementing this attack to create a Trojaned model. Once the viability of the attack was determined, several defenses were explored. These defenses included retraining the model on verified clean data to unlearn the Trojan behavior, training an autoencoder and using the reconstruction error to determine a trojaned data point or not, and lastly developing a K-Nearest-Neighbor classification technique that uses the model's last layer weights to determine if the average and range of these weights can cause the model to be classified as trojaned. This work is expounded upon in my thesis.

Main Skills: Neural Networks, Autoencoders, PyTorch
